# 更新日志

本项目采用语义化版本号。尚未发布的变化记录在“未发布”部分。

## 1.0.0 - 2026-07-19

### 新增

- IndexTTS2、VoxCPM2 与 GPT-SoVITS 的统一 Windows 创作工作台。
- 每个任务独立选择一个引擎，并提供完整原生参数及中文用途说明。
- 长文本分段、任务恢复、音频合并、项目持久化、历史记录与音频库。
- 应用内引擎安装、模型按需选择、诊断导出和 GitHub 自动更新入口。
- 舒适与紧凑两种界面密度、窗口缩放、键盘快捷键与高 DPI 布局。

### 安全与可靠性

- 引擎代码和安装工具固定版本并进行 SHA-256 校验。
- 本地音频通过受控 Electron 协议播放，渲染进程不获得任意文件读取权限。
- 项目、设置、安装状态和诊断包使用原子写入与明确失败状态。
