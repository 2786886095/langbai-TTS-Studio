# 更新日志

本项目采用语义化版本号。尚未发布的变化记录在“未发布”部分。

## 1.2.2 - 2026-07-20

### 修复

- 修复社区模型完成解压后，因 `installed\auto` 版本目录尚未创建而无法移动到最终安装位置的 Windows `[WinError 3]`。
- 增加从下载、解压、创建版本目录、原子移动到写入模型清单的完整安装回归测试。

## 1.2.1 - 2026-07-20

### 修复

- 修复社区模型下载地址跳转至 ModelScope 官方 CDN 后被误判为不受信任地址的问题；所有重定向仍会在连接前执行严格域名与 HTTPS 校验。
- VoxCPM2 训练配置改为默认占满工作区，进度、任务与完整日志改为大尺寸右侧抽屉，解决表单过小和固定侧栏浪费空间的问题。

### 验证

- 使用当前编译后端实际下载社区模型首个数据块，确认任务进入下载状态且不再触发地址拒绝。
- Windows 自动构建完成后验证后端测试、前端构建、安装包生成和打包进程清理。

## 1.2.0 - 2026-07-20

### 新增

- 左侧统一“模型训练”入口，可选择 GPT-SoVITS 或 VoxCPM2，并提供分步中文训练指引、官方文档和 B 站学习入口。
- VoxCPM2 软件内 LoRA 与全量 SFT 训练，支持本地数据清单、完整参数、日志、检查点、安全停止和恢复训练。
- GPT-SoVITS 官方训练工作台由软件检测、启动、嵌入和终止；拦截上游 `inbrowser=True`，不再弹出外部浏览器。
- 三个引擎各自独立的软件内可视化运行终端，显示启动命令、PID、日志和运行状态。
- GPT-SoVITS 社区模型支持 Hugging Face、B 站/网盘来源入口和本地权重智能识别配对。

### 改进

- 完整推理参数改为从右侧弹出的宽抽屉，释放创作区空间并改善搜索与分组操作。
- 软件更新可在全局显示明显提示，并在退出或安装更新前检测引擎、生成与训练活动。
- 仅在存在活动进程或任务时询问是否终止；关闭时对训练器执行安全停止并清理子进程。
- 修复 VoxCPM2 超长本地模型路径撑破训练布局的问题。
- 优化运行状态轮询和本地扫描，减少界面卡顿与重复请求。

## 1.1.0 - 2026-07-19

### 新增

- 按引擎独立保存、编辑、删除和应用角色声音配置。
- GPT-SoVITS 角色声音成对保存 GPT `.ckpt`、SoVITS `.pth`、参考音频、精确文本和模型版本。
- GPT-SoVITS 社区模型广场，支持作品/语言分类、搜索、安全下载、进度展示和下载后创建角色声音。

### 改进

- IndexTTS2 与 VoxCPM2 明确采用参考音频/音色设计，不再把已绑定程序误显示为通用角色权重。
- 删除界面缩放设置与缩放快捷键，保留舒适/紧凑信息密度。
- Setup 安装包使用明确文件名，并固定到标准安装目录，避免误覆盖源码仓库。
- 社区模型仅导入权重、参考音频和说明文件；拒绝越界 ZIP 路径、脚本与可执行文件。
- 长音频不再设置软件时长或项目文本上限；最终音频按分段流式写入，超过传统 WAV 容量时自动使用 RF64。

## 1.0.0 - 2026-07-19

### 新增

- IndexTTS2、VoxCPM2 与 GPT-SoVITS 的统一 Windows 创作工作台。
- 每个任务独立选择一个引擎，并提供完整原生参数及中文用途说明。
- 长文本分段、任务恢复、音频合并、项目持久化、历史记录与音频库。
- 应用内引擎安装、模型按需选择、诊断导出和 GitHub 自动更新入口。
- 舒适与紧凑两种界面密度、窗口缩放、键盘快捷键与高 DPI 布局。

### 安全与可靠性

- 引擎代码和安装工具固定版本并进行 SHA-256 校验。
- 本地音频通过受控 Electron 协议播放，渲染进程不获得任意文件读取权限。
- 项目、设置、安装状态和诊断包使用原子写入与明确失败状态。
