# 隐私说明

langbai-TTS-Studio 是本地运行的 Windows 语音生成工具。本项目不内置账号系统、广告、行为分析或遥测上报。

## 本地保存的数据

应用会在用户选择的目录以及 Windows 用户数据目录中保存：

- 语音项目、引擎参数与应用设置；
- 生成任务、分段状态、音频文件和运行日志；
- 用户主动创建的诊断包；
- 已下载的引擎代码、独立 Python 环境和用户选择的模型权重。

这些内容默认不会上传给本项目维护者。诊断包只有在用户主动导出并自行发送时才会离开本机；导出前应检查其中是否包含不希望分享的文本、路径或日志。

## 发生网络访问的场景

只有在用户触发对应功能时，应用才会访问网络：

- 从 GitHub 检查或下载应用更新；
- 从上游官方来源下载引擎代码与安装工具；
- 从模型托管站点下载用户明确选择的模型权重；
- 打开用户点击的 HTTPS 帮助、许可或项目链接。

这些服务会按各自的隐私政策处理 IP 地址、请求日志等数据。本项目无法控制第三方服务的日志保留规则。

## 用户控制

用户可以修改输出与引擎安装目录，并删除应用中提供删除入口的数据。卸载应用不会默认删除用户生成的项目、音频和已下载模型，避免误删大体积或不可恢复的数据；如需彻底清理，可按应用“设置与诊断”页面显示的实际数据目录手动处理。

## 更新

隐私行为发生实质变化时，本文件会随版本更新。安全问题请按 [SECURITY.md](SECURITY.md) 中的方式报告。
